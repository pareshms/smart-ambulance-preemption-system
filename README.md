🚑 Ambulance Management System
A real-time emergency response platform connecting patients, drivers, and hospitals.

The Ambulance Management System (AMS) is designed to reduce emergency response times. It allows patients to book the nearest available ambulance, enables drivers to navigate via GPS, and helps hospital admins manage their fleet efficiently.

🌟 Key Features
Real-time Tracking: Live GPS tracking of ambulances using Google Maps API.

Emergency Booking: One-tap booking for patients with automatic "nearest driver" assignment.

Driver Dashboard: View incoming requests, patient location, and emergency type (ICU, Oxygen, Basic).

Admin Panel: Manage ambulance fleet, monitor active trips, and verify driver registrations.

Hospital Integration: Instant notification to the receiving hospital about the patient's condition.
